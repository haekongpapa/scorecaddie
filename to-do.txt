6.스코어 입력 화면 수정
  입력화면을 2 스텝으로 구분
  
  1) 골프장 및 코스 선택 
    골프장선택, 9H/18H 선택, 
    코스선택 : 18H인 경우 전반(1~9홀)/후반(10~18홀) 각각 선택 9H인 경우 전반(1~9홀)만 선택, 
    라운드 일자, 
    날씨는 라운드 당시 날씨 표시
    스코어 카드 버튼 
       
  2) 스코어 카드 입력  
    홀 리스트를 표형태로 표시 
    전반(1~9H) 과 후반 (10~18H)을 나눠서 표시
    9H 선택한 경우는 전반(1~9H)만 표시
    홀마다 파 타수와 실제 입력한 타수 표시
    홀번호 클릭 시 스코어 입력 항목 데이터 표시    

     스코어 입력 항목
    파 규정 타수표시, 
    - 티샷결과 : 페어웨이,페어실패,페널티,OB 중 선택 
    - 페널티타수/OB타수 : 페널티 타수, OB 타수 , 벙커유무 표시
    - 온그린 핀거리 : 0m ~ 20m와 20m이상중 선택
    - 스코어 : 온그린 타수, 퍼트 타수 입력
    - 메모 : 텍스트 100자로 제한
    - 저장 버튼
