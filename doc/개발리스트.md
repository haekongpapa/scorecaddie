근거 문서: `doc/pages.md`, `doc/mockups/`, `doc/ScoreCaddie_분석설계_요약.pptx`, `app/prisma/schema.prisma`, `doc/admin-csv-upload.md`
구현 여부는 이번에 `app/src` 실제 코드를 직접 확인한 결과 기준. 범례: ✅ 완료 / 🟡 부분 구현 / ⬜ 미구현

## ⚠️ 확인 중 발견한 사항 (memory.md 이력과 실제 코드 불일치)

- `app/src/app/login/page.tsx`, `signup/page.tsx`, `page.tsx`(랜딩)는 전부 "OO 화면 (구현 예정)" 안내 문구만 있는 placeholder였음 — 예전 세션 로그(memory.md 8번)에는 "로그인/회원가입 페이지 구현 완료"로 남아있으나 실제로는 폼/버튼/`signIn()` 연동이 전혀 없는 상태.
- 골프장 공공데이터 연동 코드(`src/lib/public-data/`, `scripts/sync-golf-courses.ts`, 좌표 변환 유틸)가 프로젝트 폴더에 전혀 존재하지 않음 — `app/README.md`에 "샌드박스 임시환경에서 검증(PC에는 남지 않음)"이라 적혀 있어, 예전에 검증만 하고 실제 파일은 저장된 적이 없었던 것으로 보임. **처음부터 다시 작성 필요.**
- 아래 리스트의 "미구현" 항목들은 이 재확인을 반영한 것이며, 이전에 작성한 개발리스트(요약본)보다 범위가 넓음.
